13 Prepare : Checkpoint B - Shell Scripts

This exercise walks you through the second section of the preparation material for this week.

Overview

This checkpoint is intended to help you practice the syntax of writing shell scripts in Linux.

Instructions

For this checkpoint, you need to go through the tutorial, Writing Shell Scripts, up through (and including) "Flow Control - Part 2".

For submission, mark the sections of the tutorial you completed in the accompanying I-Learn assessment. To "Complete" a section means that you have read the text, tried out any commands that you found new or interesting, and that you have written the script to build the HTML file that the author walks you through in each section.
