13 Prove: Data Analysis - Movies

Overview

For this assignment you will be working with a dataset of movie information.

You should use Python to process this data, calculate values, and produce graphs. Then, prepare a PDF document to report on your findings and upload it to I-Learn.

Because the focus of this assignment is the result of your work, rather than creating a general purpose program, your code is not required to be as elegant as in other assignments. But keep in mind that using good style will help you keep things organized.

In addition, you are welcome to use the interactive console for elements of this if you prefer, rather than having to put all of the code in a Python program to submit in the end. However, in addition to your PDF report, please submit some evidence of the Python code you have used.

Please note that there are many ways to accomplish these tasks. You will need to search for examples and solutions online. As you do so, try to keep an eye out for best practices, etc. For example, you shouldn't iterate through each row of the Pandas dataframe to compute an average by hand. It has a built-in function to do this.

Requirements

Part I - Specific assignments

These requirements are intended to be straightforward in the expectations. It may still require research on your part to learn how to accomplish each one, but their shouldn't be ambiguity in the task itself. Please complete each one and report on it in your PDF document.

  1. Calculate the mean and median values for rating.

  2. According to this dataset, what was the first year that the "PG-13" MPAA rating was issued?

  3. For all the movies that have an MPAA Rating (e.g., "PG", "R", etc.), produce a bar chart showing the number of movies in each of these categories.

  4. Produce a graph showing the number of movies released each year.

Part II - Come up with supporting evidence

These requirements ask you to address a specific question, but require you to exercise a little bit of creativity in exactly what to produce.

  1. It might seem that some categories of movies (e.g., Action, Animation, etc.) are more likely to have "R" ratings than others. For those moves that have an MPAA Rating (e.g., "PG", "R", etc.), determine which type of movie is most "R-Rated." Produce a graph and/or explain your reasoning.

  2. It might seem that action movies, with all their special effects, cost much more than comedies to produce. Produce a graph and give an explanation to support or refute this idea.

  3. Does the type of movie (e.g., Action, Animation, etc.) fluctuate over time? Are there certain years or parts of decades that produced higher amounts of certain types of movies? Produce a graph to address this question and explain your findings.

Part III - Show Creativity

These requirements expect you to come up with your own insight.

  1. What are common characteristics of long movies?

  2. Can you find anything interesting about movies that have the highest or lowest rating?

  3. Find something else in this dataset that you consider interesting. Produce a graph to communicate your insight.
